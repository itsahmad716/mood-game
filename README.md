🎮 Mood-Based Mini Game Website

A simple interactive website where users select their mood and play a mini-game based on it.

💡 Features
Choose between 4 moods: Bored, Angry, Happy, Confused
Each mood opens a different mini-game
Built using HTML, CSS, and JavaScript
Works on both mobile and desktop browsers
🛠️ Technologies Used
HTML
CSS
JavaScript
🚀 How to Run
Download or clone the project
Open index.html in any browser
Start playing
🎯 Purpose

This project was built to practice web development and create a fun, interactive user experience using basic coding skills.
